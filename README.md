# Belize Culture & Portfolio Website

This repository contains the complete project package for CMPS1024 Assignment #1, including:
- Freelance Portfolio (leahhall.html)
- Belize Culture & Heritage pages (belize-home.html, belize-landmarks.html, belize-traditions.html)

## How to View

1. Clone or download this repository.
2. Open the folder in VS Code or your preferred editor.
3. Open any HTML file (e.g., leahhall.html) in your browser to view the page.
